# ProyectoPython

If you will run the project on your local machine, you needs install the follow commands in your Terminal:

##### Important!! If you have already installed some packages, you don't need install them.

   `pip install streamlit`

   `sudo apt-get install python3-matplotlib`

   `pip install pandas`

   `pip install plotly`

   `pip install numpy`

   `pip install Pillow`
